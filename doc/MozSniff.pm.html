NAME

MozSniff - Sniff out various versions of browsers claming to be ``Mozilla''.

SYNOPSIS

I got tired of redoing this several times, or using CGI::user_agent(), so I wrote my own.

DESCRIPTION

You should be able, I hope, to do something like: 

        use CGI::MozSniff;
        my $foo = new CGI::MozSniff;
        my $bar = $foo->sniff;

Values for $foo->sniff are:

  • navigator_$ver - Netscape Navigator v.$ver
  • communicator_$ver - Netscape Communicator v.$ver
  • msie_$ver - Microsoft Internet Explorer v.$ver
  • msie_$ver_AOL - Microsoft Internet Explorer v.$ver (AOL)
  • opera_$ver - Opera, a pretty nice browser, as far as Windoze browsers go...
  • other - Some other browser
